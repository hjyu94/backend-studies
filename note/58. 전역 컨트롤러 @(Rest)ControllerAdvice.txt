58. 전역 컨트롤러: @(Rest)ControllerAdvice

예외 처리, 바인딩 설정, 모델 객체를 모든 컨트롤러 전반에 걸쳐 적용하고 싶은 경우에 사용한다.
    ● @ExceptionHandler
    ● @InitBinder
    ● @ModelAttributes

적용할 범위를 지정할 수도 있다.
    ● 특정 애노테이션을 가지고 있는 컨트롤러에만 적용하기
    ● 특정 패키지 이하의 컨트롤러에만 적용하기
    ● 특정 클래스 타입에만 적용하기

참고
    ● https://docs.spring.io/spring/docs/current/spring-framework-reference/web.html#mvc-ann -controller-advice


[실습]

    @ControllerAdvice
    public class BaseController {
        // @ExceptionHandler
        // @InitBinder
        // @ModelAttributes
    }

    모든 컨트롤러에 위 3가지 종류가 전부 반영이 된다.

    그런데 나는 두세개의 컨트롤러는 A, 세네개의 컨트롤러는 B를 쓰고 싶다!
    이 경우를 위해서 범위를 지정할 수 있는 기능이 추가되었다.

    @ControllerAdvice(assignableTypes = {EventController.class, EventApi.class})
    public class BaseController {
        // @ExceptionHandler
        // @InitBinder
        // @ModelAttributes
    }

    이외에도 패키지 단위로, 특정 어노테이션이 붙은 컨트롤러 단위로 @ControllerAdvice를 적용할 수 있다.
